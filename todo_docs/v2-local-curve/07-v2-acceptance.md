# 06 V2 验收清单

## 数据

- [ ] 采样点数据结构可用。
- [ ] 控制点数据结构可用。
- [ ] 拟合结果数据结构可用。
- [ ] 有效时间范围可用。

## 拟合

- [ ] 多项式拟合可用。
- [ ] SmoothStep 可用。
- [ ] SmootherStep 可用。
- [ ] Logistic 可用。
- [ ] 指数趋近可用。
- [ ] 公式显示最多 2 位小数。

## 信息浮层

- [ ] 当前时间显示正确。
- [ ] 当前速度显示正确。
- [ ] 总路程显示正确。
- [ ] 有效范围显示正确。
- [ ] 可折叠和展开。

## 预览

- [ ] 运动预览可播放。
- [ ] 运动预览可暂停。
- [ ] 运动预览可重置。
- [ ] 封闭区域显示清晰。

## 皮肤资源

- [ ] 运动预览图片资源通过换皮接口获取。
- [ ] 播放、暂停、重置图标通过换皮接口获取。
- [ ] 资源缺失时能回退 Qt 默认皮肤或文本占位。
