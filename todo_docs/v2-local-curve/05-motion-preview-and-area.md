# 05 运动预览与封闭区域

## 目标

## 总路程规则

纵轴为位置时：

```text
velocity(t) = derivative(position(t))
totalDistance = integral(abs(velocity(t)), t0, t1)
```

## 注意

封闭区域不等于总路程。封闭区域只是辅助视觉表达，不能替代路程计算。
