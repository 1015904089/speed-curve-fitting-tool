# 曲线绘制与拟合工具 TODO 文档树

## 文档定位

本文档树用于把项目拆成可执行的阶段和任务。当前项目按三个版本推进：

- V1：白板 UI 原型，只做界面、绘画、滑块拖动，不涉及计算。
- V2：本地曲线能力，加入数据、速度、总路程、有效范围、本地函数族拟合和运动预览。
- V3：PySR 自动符号拟合，加入 Python Worker、后台拟合、多候选公式和环境处理。

## 推进原则

- V1 不做任何真实拟合和物理计算，只验证操作体验。
- V2 不接 PySR，先把本地曲线闭环做稳定。
- V3 再接入 PySR，避免早期被 Python、Julia、打包和耗时任务拖慢。

## 推荐开发顺序

1. 完成 V1 的窗口布局、拖拽分区、曲线区绘画和滑块拖动。
2. 完成 V2 的数据模型、采样、速度、路程、本地函数族和运动预览。
3. 完成 V3 的 PySR 后台任务、JSON 通信、结果展示和失败处理。
