# 03 PySR 拟合策略

## 目标

限制 PySR 搜索范围，让自动生成的公式尽量简洁、可读、适合运动曲线。

## 建议策略

- 先限制操作符，只允许常用函数。
- 优先生成适合运动曲线的简洁表达式。
- 不追求绝对最低误差，避免公式不可读。
- 拟合结束后由 Worker 筛选最佳结果。

## 核心指导代码

PySR 操作符方向：

```python
binary_operators = ["+", "-", "*"]
unary_operators = [
    "exp",
    "square",
    "cube",
]
```

如果需要加入 SmoothStep 或 Logistic，应优先作为受控自定义函数，而不是无限制开放复杂表达式。
