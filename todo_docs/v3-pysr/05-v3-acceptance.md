# 05 V3 验收清单

## Worker

- [ ] Python Worker 可启动。
- [ ] Worker 可读取请求 JSON。
- [ ] Worker 可写入结果 JSON。
- [ ] Worker 失败时能返回错误。
- [ ] Worker 超时时主程序可恢复。

## PySR

- [ ] PySR 可被调用。
- [ ] 操作符范围受到限制。
- [ ] 复杂度受到限制。
- [ ] 能返回最佳公式。
- [ ] 能返回候选公式。

## 主程序

- [ ] 主程序不卡顿。
- [ ] 拟合中状态显示正确。
- [ ] 成功结果显示正确。
- [ ] 失败结果显示正确。
- [ ] 用户可以取消任务。
