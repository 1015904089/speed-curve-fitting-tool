# 03 滑块区域白板

## 目标

滑块区域在 V1 中只做一维位置拖动和录制状态演示，不生成真实采样数据。

## V1 录制规则

- 鼠标按下滑块即开始记录状态。
- 鼠标松开滑块即结束记录状态。
- 当前录制时间可以用计时器显示。
- 滑块值表示位置输入。
- 曲线拟合区域的 Y 轴表示速度，不直接等同于滑块位置。
- 不计算速度。
- 不生成拟合曲线。

## 核心指导代码

滑块是一维位置输入，V1 只需要保留状态：

```cpp
struct SliderDraftState {
    bool recording;
    double elapsedSeconds;
    double normalizedPosition; // 0.0 到 1.0
};
```

`normalizedPosition` 后续会根据位置范围映射为真实位置。速度曲线在 V2 之后由位置采样计算得到。
