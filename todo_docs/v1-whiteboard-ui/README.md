# V1 白板 UI 原型

## 目标

V1 只做界面、绘画、缩放、平移、滑块拖动和占位显示，不涉及任何计算。

验收目标：

```text
用户打开软件后，可以拖动布局、缩放网格、画线、拖滑块。
但不要求生成公式，也不要求计算速度、路程、误差。
```

## V1 边界

- 不计算速度。
- 不计算总路程。
- 不计算有效范围。
- 不生成真实公式。
- 不接本地拟合。
- 不接 PySR。
- 不制作自定义皮肤，默认使用 Qt 默认皮肤。
