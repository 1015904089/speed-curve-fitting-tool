# 01 布局与可拖拽分区

## 目标

实现三块主区域：

- 左上：曲线拟合区域
- 左下：滑块区域
- 右侧：结果展示区域

所有区域支持拖拽调整尺寸。

## 默认比例

- 左侧操作区：70%
- 右侧结果展示区：30%
- 曲线拟合区域：左侧高度的 65%
- 滑块区域：左侧高度的 35%

## 核心指导代码

Qt 中可以使用嵌套 splitter 表达布局方向：

```cpp
// 主方向：左右分割
// 左侧方向：上下分割
//
// MainSplitter(Horizontal)
//   - LeftSplitter(Vertical)
//       - CurveCanvas
//       - SliderPanel
//   - ResultPanel
```

保存布局时只保存分隔比例或 splitter state，不保存绝对像素，避免不同屏幕尺寸下恢复异常。
