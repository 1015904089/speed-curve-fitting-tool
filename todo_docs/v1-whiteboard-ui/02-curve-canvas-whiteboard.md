## 核心指导代码

画布需要明确屏幕坐标和世界坐标的转换方向：

```cpp
struct ViewTransform {
    double scaleX;
    double scaleY;
    double offsetX;
    double offsetY;
};

// screenX = time * scaleX + offsetX
// screenY = offsetY - position * scaleY
```

V1 不需要真实拟合，但必须先把坐标转换方向固定下来。后续 V2 的曲线绘制、采样点显示和运动预览都会依赖它。
