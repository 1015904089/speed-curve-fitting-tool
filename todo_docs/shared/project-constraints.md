# 项目约束与术语

## 基础约束

- 主程序使用 C++。
- 界面建议使用 Qt。
- 项目目录指定为 `D:\drawplot`。
- Qt 目录指定为 `D:\qt6`。
- 文档目录放在 `D:\drawplot\todo_docs`。
- 界面采用三块主区域：左上曲线拟合区域、左下滑块区域、右侧结果展示区域。
- 每个区域都要支持拖拽调整尺寸。
- 非网格界面中涉及图片、图标、装饰图、预览占位图等视觉资源时，必须预留换皮接口。
- 默认皮肤使用 Qt 默认皮肤，不额外强制自定义视觉主题。
- 网格绘制属于曲线功能本体，不纳入图片换皮接口管理。
- 滑块是一维运动。
- 横轴是时间，单位为秒。
- 滑块区域记录的是位置输入。
- 曲线拟合区域的 Y 轴固定为速度。
- 当前时间、当前速度、总路程、有效范围显示在曲线拟合区域。
- 滑块区域只显示当前录制时间和滑块位置。
- 速度范围不作为显示项。
- 曲线区信息显示采用类似 Blender 的可折叠轻量浮层。

## 术语

- 曲线拟合区域：左上主绘图区，负责网格、绘画、控制点、曲线和信息浮层。
- 滑块区域：左下一维拖动区域，负责位置录制交互。
- 结果展示区域：右侧区域，负责公式、参数、误差、预览和操作状态。
- 信息浮层：曲线拟合区域内的可折叠数据显示层，显示当前时间、当前速度、总路程、有效范围。
- 有效范围：当前曲线或录制数据在时间轴上的起止范围。
- 总路程：后续根据速度绝对值随时间积分后的结果。

## 版本边界

### V1 不做

- 不做真实拟合。
- 不计算速度。
- 不计算总路程。
- 不计算有效范围。
- 不接 PySR。
- 不做真实公式生成。
- 不制作自定义皮肤，只预留换皮接口。

### V2 不做

- 不接 PySR。
- 不做多候选符号公式。
- 不做 Python/Julia 环境检测。

### V3 才做

- Python Worker。
- PySR 调用。
- 自动符号回归。
- 多候选公式展示。
- 超时、失败、环境缺失处理。
