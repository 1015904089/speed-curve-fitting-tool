# 验收总表

## V1 白板 UI 原型

- [ ] 窗口打开后显示三块主区域。
- [ ] 左上为曲线拟合区域。
- [ ] 左下为滑块区域。
- [ ] 右侧为结果展示区域。
- [ ] 左右分隔条可拖动。
- [ ] 上下分隔条可拖动。
- [ ] 各区域有最小尺寸。
- [ ] 双击分隔条可恢复默认布局。
- [ ] 布局比例可保存和恢复。
- [ ] 曲线区显示常驻网格。
- [ ] 曲线区支持缩放和平移。
- [ ] 曲线区支持鼠标绘画。
- [ ] 曲线区支持控制点占位。
- [ ] 曲线区有可折叠信息浮层占位。
- [ ] 滑块可拖动。
- [ ] 鼠标按下滑块进入录制状态。
- [ ] 鼠标松开滑块退出录制状态。
- [ ] 滑块区显示当前录制时间。
- [ ] 右侧结果区显示占位内容。

## V2 本地曲线能力

- [ ] 控制点和采样点有正式数据模型。
- [ ] 滑块录制能生成一维位置采样数据。
- [ ] 曲线区能显示当前时间。
- [ ] 曲线区能显示当前速度。
- [ ] 曲线区能显示总路程。
- [ ] 曲线区能显示有效范围。
- [ ] 信息浮层可折叠和展开。
- [ ] 支持低阶多项式拟合。
- [ ] 支持 SmoothStep。
- [ ] 支持 Logistic。
- [ ] 支持指数趋近。
- [ ] 支持运动预览。
- [ ] 支持封闭区域高亮。
- [ ] 公式小数最多显示 2 位。

## V3 PySR 自动拟合

- [ ] C++ 能启动 Python Worker。
- [ ] C++ 能发送采样点 JSON。
- [ ] Python Worker 能调用 PySR。
- [ ] PySR 拟合在后台运行。
- [ ] 主界面不会因 PySR 卡顿。
- [ ] 能返回最佳公式。
- [ ] 能返回拟合误差。
- [ ] 能返回公式复杂度。
- [ ] 能展示多候选公式。
- [ ] 能处理超时。
- [ ] 能处理环境缺失。
- [ ] 能处理拟合失败。
